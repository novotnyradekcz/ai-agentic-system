# Place your data files here

## Supported formats:

### PDFs:
- *.pdf

### Audio files:
- *.mp3
- *.wav
- *.m4a
- *.flac
- *.ogg
- *.wma

### Video files (audio extraction):
- *.mp4

## Example structure:
```
data/
├── research_paper.pdf
├── lecture_recording.mp3
├── notes.pdf
└── interview.wav
```

Add your files here and run:
```bash
python main.py
```
