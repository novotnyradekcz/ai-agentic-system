# Generated Content Output Directory

This directory stores all content generated by the AI Agent:

## File Types

- **HTML Pages**: `webpage_*.html` - Complete, styled web pages
- **PDF Documents**: `*.pdf` - Professional documents in various styles (report, guide, tutorial, whitepaper)

## File Naming Convention

Files are automatically named with:
- Content type
- Topic (sanitized)
- Timestamp

Examples:
- `webpage_neural_networks_20260202_143022.html`
- `machine_learning_guide_20260202_143022.pdf`

## Opening Files

- **HTML files**: Can be opened directly in your web browser. They are fully self-contained with embedded CSS styling.
- **PDF files**: Can be opened with any PDF viewer.

## Notes

- Files are automatically created when using HTML or PDF generation tools
- All files include metadata (timestamp, topic, style)
- This directory is created automatically if it doesn't exist
