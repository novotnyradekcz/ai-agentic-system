# Generated Content Output Directory

This directory stores all content generated by the AI Agent:

## File Types

- **Blog Posts**: `blog_post_*.txt`
- **Newsletters**: `newsletter_*.txt`
- **HTML Pages**: `webpage_*.html`

## File Naming Convention

Files are automatically named with:
- Content type
- Style (for blog posts)
- Topic (sanitized)
- Timestamp

Example: `blog_post_professional_machine_learning_20260201_143022.txt`

## Opening HTML Files

HTML files can be opened directly in your web browser. They are fully self-contained with embedded CSS styling.

## Notes

- Files are automatically created when using content generation tools
- All files include metadata (timestamp, topic, style)
- This directory is created automatically if it doesn't exist
